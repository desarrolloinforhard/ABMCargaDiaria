# Frontend Desktop

Cliente desktop Python para ABMCargaDiaria.

Responsabilidades:

- UI Tkinter + ttkbootstrap + TkInforHard.
- Pantallas, formularios, grillas y dashboard.
- Consumo futuro de la API backend.
- Sin conexion directa a ASA9.

## Ejecutar

```powershell
cd frontend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python -m app.main
```
