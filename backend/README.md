# Backend API

API Node.js + Express para ABMCargaDiaria.

Responsabilidades futuras:

- Exponer contratos REST para el frontend desktop.
- Centralizar reglas de negocio compartidas.
- Conectar ASA9 / Sybase SQL Anywhere mediante ODBC 32-bit.
- Procesar CSV bancarios.
- Manejar logs, errores, auditoria y credenciales fuera del cliente desktop.

## Ejecutar

```powershell
cd backend
npm install
npm run dev
```
